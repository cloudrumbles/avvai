# Sample Tamil Text for Sandhi Checking
# This file contains various Tamil sentences for testing

# Correct sandhi examples
தமிழ் மொழி சிறந்தது.
வணக்கம்! எப்படி இருக்கிறீர்கள்?
நல்ல புத்தகம் படித்தேன்.

# Incorrect sandhi (vallinam should double)
பாட்டு பாடினான்.
வீடு கட்டினான்.
பள்ளி செல்கிறான்.

# Mixed correct and incorrect
அவன் நல்ல பாட்டு பாடினான்.
நாம் தமிழ் படிக்கிறோம்.
